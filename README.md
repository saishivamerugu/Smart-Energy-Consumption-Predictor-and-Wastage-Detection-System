# ⚡ Smart Energy Consumption Predictor & Wastage Detection System

## 📌 Overview

Smart Energy Consumption Predictor & Wastage Detection System is a Machine Learning-based application that analyzes historical electricity usage data to forecast future energy consumption and identify potential energy wastage. The system helps users make data-driven decisions for optimizing energy usage, reducing operational costs, and improving overall energy efficiency.

## 📈 Machine Learning Models Used

### Linear Regression

Used as a baseline model for predicting energy consumption.

### Random Forest Regressor

Captures complex relationships and improves prediction accuracy.

### XGBoost Regressor

Provides high-performance gradient boosting for accurate forecasting.

---

## 📉 Model Evaluation Metrics

The models were evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

The best-performing model was selected for deployment.
